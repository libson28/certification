Comment ça marche !

Découvrez les differentes parties pour la création de votre compte


Etape 1

Une fois sur la page d'accueil de la plateforme, cliquer sur le bouton “Se connecter ” qui va vous rediriger sur la page de connexion. Renseigner ces champs pour vous connecter; sinon, si vous ne disposez pas de compte, suivez l'étape 2

Etape 2

Etape 2

Cliquer sur le lien “inscrivez-vous” pour aller vers la page d'incription. Mettez les informations demandées selon le label des champs, ensuite cliquez sur “Enregistrer” s'assurer que vos informations sont bien ajoutées et que vous disposez maintenant d'un compte

Etape 3

Maintenant que vous vous êtes connectés, vous avez la possibilité de mofifier votre profil à chaque fois que cela vous chante. Vous pouvez également ajouter une photo de profil
